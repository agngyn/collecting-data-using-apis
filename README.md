# Collecting Data Using APIs

## Overview
This project demonstrates how to collect and analyze job postings data using a public Jobs API and Python.

## Objectives
- Retrieve job postings data from an API
- Count the number of job postings by technology
- Count the number of job postings by location
- Store results for analysis

## Tools Used
- Python
- Requests
- Pandas

## Files
- `Jobs_API.ipynb`: Jupyter Notebook containing the API data collection and analysis

## How to Run
Open the notebook and run all cells in order.
